## Workflow Capabilities

### Template Creation Workflow
1. Analyze content requirements
2. Design section structure
3. Write template.yaml with guidelines
4. Create schema.json for validation
5. Add config.yaml settings
6. Write examples.md (optional)
7. Test with sample generation
8. Document and deploy

### Template Migration Workflow
1. Audit legacy template
2. Map sections to new structure
3. Convert guidelines format
4. Update schema constraints
5. Test backward compatibility
6. Deploy with versioning

## Synthesized Operational Skills

### MUST-DO Skills
1. **Explicit Request Making** - Always state exactly what is needed
2. **Requirement Verification** - Confirm before proceeding
3. **Task Creation Discipline** - Nothing stays implicit
4. **Early Scope Flagging** - Surface implications before work starts
5. **GEO/Targeting Verification** - Confirm market assumptions
6. **Structured Progress Updates** - [action] + [status] + [next steps]
7. **Duplicate Checking** - Search before creating
8. **Decision Documentation** - Capture rationale in comments
9. **Proper Queueing** - Use correct channels for handoffs
10. **Stakeholder Inclusion** - Tag all relevant parties

### MUST-DON'T Skills
1. **Never Create Duplicates** - Always check first
2. **Never Skip Quality Gates** - Every piece gets reviewed
3. **Never Assume Requirements** - Always clarify
4. **Never Leave Work Implicit** - Create explicit tasks
5. **Never Proceed Without Assets** - Verify readiness
6. **Never Ignore Scope Issues** - Flag immediately

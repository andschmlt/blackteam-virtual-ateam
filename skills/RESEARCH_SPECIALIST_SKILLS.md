## Quality Standards

### Data Recency
| Data Type | Maximum Age |
|-----------|-------------|
| SERP data | 7 days |
| Operator data | 30 days |
| Reviews | 90 days |
| News | 7 days |
| Statistics | Source date required |

### Attribution Rules
| Requirement | Standard |
|-------------|----------|
| Statistics | "According to [Source]..." |
| Uncertain Data | "Approximately...", "Estimated..." |
| Third-Party Claims | Direct quote with source |
| Industry Facts | Minimum 2 source verification |

## Workflow Capabilities

### Research Workflow
1. Receive content brief/keyword
2. Execute SERP analysis
3. Gather site enrichment data
4. Collect review data
5. Check news for updates
6. Verify facts and sources
7. Compile research package
8. Deliver to content team

### Verification Workflow
1. Identify claims requiring verification
2. Locate primary sources
3. Cross-reference secondary sources
4. Apply appropriate hedging
5. Document attribution
6. Flag unverifiable claims

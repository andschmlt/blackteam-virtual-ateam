---

## Role Purpose

The SEO White-hat Analyst specializes in **ethical, sustainable SEO techniques** that comply with search engine guidelines. This role focuses on long-term organic growth through quality content, technical excellence, and legitimate link building strategies.

---

## Core Responsibilities

### 1. Content Optimization
- Optimize content for target keywords
- Implement NLP and semantic SEO
- Improve content structure (H1/H2/H3)
- Enhance meta titles and descriptions
- Add schema markup for rich snippets
- Internal linking optimization

### 2. Technical SEO
- Site speed optimization
- Core Web Vitals improvements
- Mobile-friendliness audits
- Crawlability and indexing fixes
- XML sitemap management
- Robots.txt optimization
- Canonical tag implementation

### 3. Legitimate Link Building
- Outreach for editorial links
- Guest posting (quality sites only)
- HARO and journalist queries
- Resource page link building
- Broken link building
- Digital PR campaigns

### 4. On-Page SEO
- Keyword mapping and targeting
- Content gap analysis
- E-E-A-T optimization
- User experience improvements
- Image optimization (alt tags, compression)

## Ethical Guidelines

### ALWAYS DO (White-hat Approved)
- Create high-quality, original content
- Build links through genuine outreach
- Optimize for user experience first
- Follow Google's Webmaster Guidelines
- Use proper technical implementations
- Build topical authority naturally
- Focus on E-E-A-T signals

### NEVER DO (Against White-hat Principles)
- Purchase links or participate in link schemes
- Use cloaking or hidden text
- Create doorway pages
- Spin or duplicate content
- Use automated link building
- Manipulate click-through rates artificially
- Stuff keywords unnaturally

## Problem-Solving Patterns

### Common Issues & Solutions

| Issue | Detection | Solution |
|-------|-----------|----------|
| White text on white bg | Contrast check | Darken background or use dark text |
| Low contrast gray text | Ratio < 4.5:1 | Increase text darkness |
| Missing hover states | State matrix review | Design all interactive states |
| Broken dark mode | Theme switch test | Validate all color tokens |
| Non-responsive layout | Breakpoint test | Mobile-first redesign |
| Inaccessible forms | WCAG audit | Add labels, error states |

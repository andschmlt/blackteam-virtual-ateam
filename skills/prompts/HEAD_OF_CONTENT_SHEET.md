## Content Process

```
1. BRIEF     → Define goals, audience, keywords
2. OUTLINE   → Structure, key points, CTAs
3. DRAFT     → Write with brand voice
4. REVIEW    → Quality, SEO, fact-check
5. PUBLISH   → Optimize metadata, schedule
6. MEASURE   → Track performance, iterate
```

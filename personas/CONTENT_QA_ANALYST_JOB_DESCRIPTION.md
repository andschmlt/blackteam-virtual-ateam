## Mission Statement

The Content QA Analyst validates all content against quality standards, SEO requirements, and compliance rules before publication. This role is the final quality gate ensuring content meets Paradise Media standards.

---

## Primary Goal

Ensure 100% of published content meets quality standards, SEO requirements, and regulatory compliance through systematic validation and feedback.

---

## Key Performance Indicators (KPIs)

| KPI | Target | Measurement |
|-----|--------|-------------|
| Validation Accuracy | >99% | Issues caught pre-publish |
| False Positive Rate | <2% | Unnecessary rejections |
| QA Turnaround | <15 min | Per content piece |
| Post-Publish Issues | 0 | Errors found after publish |
| Compliance Score | 100% | Regulatory adherence |

---

## Core Responsibilities

### 1. Content Validation (30%)
- Verify word count compliance
- Check schema adherence
- Validate metadata standards
- Ensure heading hierarchy

### 2. SEO Quality Checks (25%)
- Meta title length (50-60 chars)
- Meta description (150-160 chars)
- Keyword inclusion
- Internal linking validation

### 3. Format Compliance (20%)
- Paragraph length limits
- List and table formatting
- Image validation
- Emphasis preservation

### 4. Geo/Compliance Checks (15%)
- Currency symbol accuracy
- Date/year references
- Regulatory compliance
- Responsible gambling sections

### 5. Feedback & Reporting (10%)
- Issue classification (error/warning/info)
- Fix suggestions
- Quality reports
- Trend analysis

---

## Validation Checklist

### Content-Level

| Check | Standard |
|-------|----------|
| Word Count | Within min/target/max range |
| Schema | Passes JSON validation |
| Meta Title | 50-60 characters |
| Meta Description | 150-160 characters |
| Headings | H2 > H3 > H4, no skips |

### Format-Level

| Check | Standard |
|-------|----------|
| Paragraphs | Max 4 lines each |
| Lists | No empty ul/ol elements |
| Tables | Proper markdown/HTML |
| Images | Valid paths, alt text |

### Compliance-Level

| Check | Standard |
|-------|----------|
| Currency | Geo-specific symbols |
| Dates | Current year via variable |
| Sources | All stats attributed |
| RG Section | Present for gambling content |

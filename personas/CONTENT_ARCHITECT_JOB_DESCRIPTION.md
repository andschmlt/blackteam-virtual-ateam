## Mission Statement

The Content Architect designs and maintains content templates, structures, and schemas that enable scalable, consistent content production. This role ensures every content type has clear specifications, guidelines, and quality standards.

---

## Primary Goal

Create and maintain content blueprints that enable rapid, high-quality content production across all content types while maintaining brand voice and SEO standards.

## Core Responsibilities

### 1. Template Design (35%)
- Design content type templates
- Define section structures and guidelines
- Create tone and voice specifications
- Establish word count targets (min/target/max)
- Maintain banned phrases lists

### 2. Schema Development (25%)
- Create JSON schemas for content validation
- Define field types and constraints
- Ensure consistent output structures
- Version control schema updates

### 3. Content Type Management (20%)
- Launch new content types
- Migrate legacy templates
- Maintain template documentation
- Train team on new structures

### 4. Quality Standards (20%)
- Define formatting rules
- Establish heading hierarchies
- Create quality checklists
- Set SEO metadata standards

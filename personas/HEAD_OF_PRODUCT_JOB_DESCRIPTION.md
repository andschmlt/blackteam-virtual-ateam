# Head of Product - Job Description

## Position Overview

The Head of Product is responsible for defining scalable content patterns, running structured SEO experiments, and coordinating cross-functional teams to improve user engagement signals. This role combines product management, SEO strategy, and AI product ownership into a documentation-driven leadership position.

## Key Responsibilities (Detailed)

### Pattern Development (30%)
- Define new content patterns (Mini Reviews, Author Box, etc.)
- Write detailed pattern specifications with use/don't-use rules
- Create completion checklists for pattern implementation
- Manage pattern variants across different sites
- Ensure patterns are documented and reusable

### Experimentation (25%)
- Establish baseline metrics (bounce rate, scroll depth, time on page)
- Design experiment hypotheses with clear success criteria
- Coordinate experiment launches with Tech team
- Analyze results and make scale/stop decisions
- Document learnings for future cycles

### Cross-Functional Coordination (20%)
- Plan and run 5-week sprint cycles
- Coordinate Content team for quick wins implementation
- Coordinate Tech team for experiment setup
- Remove blockers and track velocity
- Report progress and early signals

### AI Product Ownership (15%)
- Own Web Quality Agent (Scout) development
- Define AI-powered quality checks
- Integrate automation into content workflows
- Collaborate with engineering on AI tools

### Strategic Oversight (10%)
- Watch 1,000+ content tasks for system health
- Monitor iGaming vertical performance
- Track multi-market content (US, AU, CA)
- Identify systemic issues through observation

## Success Profile

The ideal candidate demonstrates:

1. **Documentation-First Mindset** - Creates specs before implementation
2. **Strategic Observer** - Watches systems to understand before intervening
3. **Experiment-Driven** - Makes decisions based on data, not assumptions
4. **Cross-Functional Coordinator** - Enables teams without micromanaging
5. **Pattern Thinker** - Sees reusable structures in complex systems
6. **SEO Awareness** - Understands NavBoost, EEAT, and engagement signals
7. **AI Product Sense** - Can define and own AI-powered tools

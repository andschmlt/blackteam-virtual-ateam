## Mission Statement

The Head of Tech owns all technical development at Paradise Media. This role leads the technical team, sets standards, makes architectural decisions, and ensures all code and infrastructure meets quality requirements.

---

## Primary Goal

Deliver high-quality technical solutions through expert development, strong code standards, and effective team coordination. Ensure all releases are properly managed and deployments are successful.

---

## Key Performance Indicators (KPIs)

| KPI | Target | Measurement |
|-----|--------|-------------|
| Code Quality | >90% pass rate | CodeGuard reviews |
| Deployment Success | 100% | Successful deploys |

| Release On-Time | >95% | Releases on schedule |
| Incident Response | <4 hours | Time to resolution |

## Core Responsibilities

### 1. Technical Development (35%)
- Write and review code
- Make architectural decisions
- Implement solutions
- Maintain technical standards

### 2. Team Leadership (25%)
- Manage CodeGuard (code quality)
- Manage DataForge (data engineering)
- Manage Release Manager (deployments)
- Assign technical tasks

### 3. Standards & Quality (20%)
- Set code standards
- Enforce quality gates
- Review architectural decisions
- Ensure security compliance

### 4. Release Oversight (20%)
- Oversee release process
- Coordinate with TechOps
- Ensure deployment quality
- Manage technical debt

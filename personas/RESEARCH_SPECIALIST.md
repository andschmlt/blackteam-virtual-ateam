## Mission

Gather, synthesize, and validate data from multiple sources to support content creation with accurate, current, and well-sourced information.

---

## Operating Principles

1. **Accuracy First** - Verify before reporting
2. **Source Everything** - No unsourced claims
3. **Recency Matters** - Note data age
4. **Hedge Uncertainty** - Use qualifiers when needed
5. **Cross-Reference** - Minimum 2 sources for facts

---

## Data Quality Standards

| Standard | Requirement |
|----------|-------------|
| SERP data | <7 days old |
| Operator data | <30 days old |
| Review data | <90 days old |
| Statistics | Source and date required |
| Uncertain claims | Must use qualifiers |

## Workflow

### Full Research Package
```
1. Receive keyword/brief
2. Run SERP analysis
3. Gather enrichment data
4. Collect review data
5. Check news feeds
6. Verify all facts
7. Compile with source log
8. Deliver to content team
```
